  <p><strong>Minecraft Version</strong>: 1.21.11<br>
  <strong>Fabric Version</strong>: 0.18.4<br>
  <strong>Mod Version</strong>: 1.0.0</p>

  <p>Mojang added the coolest mob ever and made it a two seat horse that can dash instead of jump. This mod adds the functionality that the camel should have had from day one.</p>

  <h2>Features</h2>
    <strong>• Camel Armor</strong> – Equip camels with protective armor for added durability.<br>
    <strong>• Camel Chests</strong> – Add storage to camels for carrying items while exploring.
